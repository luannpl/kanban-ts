# Kanban-Ts

## Sobre

**Esse projeto trata-se da simulção de um quadro Kanban desenvolvido com React e TypeScript, além de Tailwind CSS para estilização.**

## Instalação
### 1º Clone o repositório.

**HTTPS**

```bash
git clone https://github.com/luannpl/kanban-ts.git
```
ou

**SSH**

```bash
git clone git@github.com:luannpl/kanban-ts.git
```

### 2º Entre no diretório

**No terminal**

```
cd kanban-ts
```

### 3º Instale as dependências necessárias

**Ainda no terminal**

```
npm install
```

### 4º Execute o projeto

```
npm run dev
```

**Após executar, pressione a tecla ctrl e clique no link que vai estar aparecendo**

## Tecnológias Utilizadas

- **React**
- **TypeScript**
- **Tailwind**
- **Shadcn**
- **React Beautiful DnD**
